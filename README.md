# RPG-character-creation
# About my project

I created a program that lets me build and manage characters for a game. I allow users to create characters with a name, class, and automatically calculated stats like strength, magic, and health. I give each character default gold and equipment, and I assign a unique backstory based on their class. I also wrote functions so I can save a character’s data to a text file, load it back, display their information, and level up the character to increase their stats. Overall, I organize all character-related data into dictionaries and handle basic file operations to store and retrieve characters.
